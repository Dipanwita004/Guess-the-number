# 🎯 Number Guessing Game

## 📌 Description
This is a simple number guessing game where the user has to guess a number within a given range and limited attempts.

## 🚀 Features
- Random number generation
- Limited attempts
- Hint system (Too high / Too low)
- User-friendly interaction

## 🛠️ Technologies Used
- Html,CSS,JavaScript

## ▶️ How to Run
1. Clone the repository
2. Run the file:
   guess-the-number.js

## 📷 Example
Guess a number between 1 and 100
Attempts left: 6
